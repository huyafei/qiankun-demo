# qiankun-demo
微前端 qiankun 库 vue2.x 使用示例

## 安装
```shell
# 安装全部
npm run install-all
# 单独安装
# 主应用项目安装
npm run install:main
# 微应用 history 模式项目安装
npm run install:history
# 微应用 hash 模式项目安装
npm run install:hash
```
## 运行
```shell
# 运行全部
npm run serve-all
# ... 同理
```

## 编译
```shell
# 编译全部
npm run build-all
# ... 同理
```
